# CardLobby

CardLobby frontend and admin portal for catalog browsing, auth, and API key request workflow.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Required Environment Variables

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `ADMIN_USER_IDS`
- `CARDHQ_ISSUER_URL`
- `CARDHQ_ISSUER_SECRET`
